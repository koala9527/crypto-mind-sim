# 策略管理指南

## 概述

NeoTrade AI 支持 AI 驱动的自动交易策略。用户可从预设模板创建策略，也可完全自定义。每个用户同一时间只能有一个激活策略。

---

## 预设策略模板

系统提供 6 种预设模板，仅提供名称、描述和提示词，交易对/模型/杠杆由用户自行选择：

| 模板 | 风格 | 推荐杠杆 | 适合场景 |
|------|------|----------|----------|
| 激进赌狗型 | 高风险高回报 | 20x | RSI 超买超卖快进快出 |
| 稳健波段型 | 中等风险 | 3x | EMA 金叉死叉，3% 止损 5% 止盈 |
| 网格套利型 | 低风险 | 2x | 震荡行情高抛低吸 |
| 趋势跟随型 | 中高风险 | 5x | MACD 趋势识别，移动止损 |
| 突破交易型 | 中高风险 | 5x | 关键价位突破交易 |
| 均值回归型 | 中等风险 | 3x | 布林带超买超卖回归 |

> 重置策略到默认值时，只恢复描述和提示词，保留用户选择的交易对、模型等参数。

## 支持的交易对

BTC/USDT、ETH/USDT、BNB/USDT、SOL/USDT、XRP/USDT、ADA/USDT、DOGE/USDT、DOT/USDT、AVAX/USDT、LINK/USDT

## 策略 API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/strategies/presets` | 获取预设模板列表 |
| POST | `/api/strategies` | 创建策略 |
| GET | `/api/strategies` | 获取用户策略列表 |
| PUT | `/api/strategies/{id}` | 更新策略 |
| POST | `/api/strategies/{id}/activate` | 激活策略 |
| POST | `/api/strategies/{id}/reset` | 重置为预设默认值 |
| GET | `/api/models` | 获取支持的 AI 模型列表 |

> **路由顺序**：`/presets` 必须在 `/{strategy_id}` 之前注册，否则会被路径参数捕获。

---

## 策略数据模型

`PromptConfig` 表字段：

| 字段 | 说明 |
|------|------|
| name | 策略名称 |
| description | 策略描述 |
| prompt_text | 提示词内容 |
| symbol | 交易对 |
| ai_model | AI 模型（默认 claude-4.5-opus） |
| leverage | 杠杆倍数 |
| execution_interval | 执行间隔（60-600 秒） |
| is_active | 是否激活 |
| user_id | 所属用户 |

---

## 使用流程

1. **创建策略**：从预设模板选择或自定义，设置交易对、AI 模型、杠杆、执行频率
2. **自定义提示词**：根据需要修改 AI 提示词内容
3. **激活策略**：点击激活，系统开始按设定频率自动执行
4. **监控执行**：通过 AI 决策日志和交易历史查看执行情况
5. **调整优化**：根据执行结果调整策略参数

---

## 执行机制

策略执行器 (`backend/engine/strategy_executor.py`) 工作流程：

1. 按 `execution_interval` 定时触发
2. 获取当前市场数据（价格、技术指标）
3. 从 User 表读取 `ai_api_key` 调用 AI 模型
4. 解析 AI 返回的 JSON 决策
5. 执行交易操作：
   - **开仓**：计算保证金、检查余额、创建 Position + Trade 记录
   - **平仓**：计算盈亏、返还保证金 + 盈亏、关闭持仓
   - **观望**：仅记录决策日志
6. 前端每 3 秒自动刷新数据

---

## 最佳实践

- 新手建议从「稳健波段型」+ Claude 4.5 Opus 开始
- 先用小杠杆测试策略效果，再逐步调整
- 定期查看 AI 决策日志，了解 AI 的推理逻辑
- 同一预设模板可搭配不同交易对和模型使用
