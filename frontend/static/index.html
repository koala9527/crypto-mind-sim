📊 关于 CryptoMindSim

✨ 项目简介

CryptoMindSim 是一个基于人工智能的加密货币模拟交易平台，旨在为用户提供零风险的交易学习环境。通过模拟真实市场行情，结合AI智能决策系统，帮助用户深入理解数字货币交易机制。

🎯 核心功能

  • • 实时行情模拟 - 基于真实交易所数据
  • • 杠杆交易体验 - 支持1-20倍杠杆
  • • AI智能策略 - 可配置多种AI交易策略
  • • 全球排行榜 - 与其他用户竞技交易能力
  • • 完整交易记录 - 持仓管理与历史追溯
  • • 风险控制系统 - 自动爆仓保护机制

🛡️ 安全保障

本平台为纯模拟环境，不涉及任何真实资金交易。所有交易均为虚拟操作，用户可以放心学习和实验各种交易策略，无需担心资金损失。

🚀 技术栈

• 后端：FastAPI + Python
• 前端：Tailwind CSS + Chart.js
• 数据库：SQLite + SQLAlchemy
• 行情：CCXT 统一接口
• AI：Claude API 决策引擎
• 部署：Docker 容器化

❓ 常见问题

这个平台是真实交易吗？ ▼

不是。CryptoMindSim 是一个100% 模拟交易平台，所有交易均为虚拟操作，不涉及任何真实资金。行情数据虽然基于真实交易所，但所有交易行为都在本地数据库中进行，完全无风险。

什么是杠杆交易？ ▼

杠杆交易是指用较小的保证金控制更大价值的资产。例如使用 5 倍杠杆，用 1000 USDT 可以交易价值 5000 USDT 的 BTC。杠杆会同时放大盈利和亏损，需要谨慎使用。

什么时候会触发爆仓？ ▼

当持仓的未实现亏损达到保证金的 90% 时，系统会自动强制平仓（爆仓），以防止亏损超过保证金。高杠杆下价格波动幅度较小就可能触发爆仓，请合理控制杠杆倍数。

做多和做空有什么区别？ ▼

做多（LONG）：看涨时使用，价格上涨则盈利，价格下跌则亏损。
做空（SHORT）：看跌时使用，价格下跌则盈利，价格上涨则亏损。

AI 策略是如何工作的？ ▼

平台集成了基于 Claude AI 的决策引擎，可以根据配置的策略提示词（Prompt）分析市场数据，生成交易建议（BUY/SELL/HOLD）。您可以在后台配置不同的策略，系统会定期执行分析并记录决策日志。

如何查看我的历史交易？ ▼

在主面板切换到"交易历史"标签页，可以查看所有已完成的交易记录，包括开仓、平仓、爆仓等操作，以及每笔交易的盈亏情况。

资金用完了怎么办？ ▼

如果您的账户余额归零且无持仓，可以联系管理员重置账户，或者注册一个新账户重新开始。这是模拟平台的优势 —— 可以无限次尝试，不断优化交易策略。

数据多久更新一次？ ▼

• 价格行情：每 60 秒从交易所拉取最新数据
• 持仓盈亏：实时计算并更新
• 排行榜：每 3 秒刷新一次
• 页面数据：登录后每 3 秒自动刷新
