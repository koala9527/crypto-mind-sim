# CryptoMindSim

面向学习和演示场景的加密货币模拟交易项目。

CryptoMindSim 使用真实市场行情做模拟盘演示，帮助用户理解开仓、持仓、平仓、手续费、爆仓风险与策略执行流程，不涉及真实资金交易。

- GitHub：`https://github.com/koala9527/crypto-mind-sim`

## 项目特点

- 模拟交易，不连接真实资金账户
- 支持多空、杠杆、手续费和爆仓风险展示
- 内置 6 种中文策略模板，可直接编辑
- 支持查看交易记录、AI 决策记录和提示词历史
- AI 配置与交易参数按用户保存
- 默认使用 SQLite，适合本地运行和开源演示

## 适合谁

- 想学习交易系统基本流程的用户
- 想做 AI 模拟交易演示的开发者
- 想快速搭建一个可运行样例项目的作者

## 快速开始

### 1. 安装依赖

```bash
uv sync
```

### 2. 准备环境变量

```bash
cp config/.env.example .env
```

推荐保留默认配置，仅按需修改 `SECRET_KEY`。

### 3. 启动项目

```bash
uv run .\main.py
```

启动后访问：

- 首页：`http://127.0.0.1:8010`
- API 文档：`http://127.0.0.1:8010/docs`

## 使用说明

- 注册后，系统会自动创建一条默认策略
- 用户通过编辑这条策略完成模板切换、自定义提示词和启停管理
- 可在“设置”中修改 AI Key、手续费率、爆仓阈值和初始模拟资金
- 如仓位触发爆仓，系统会自动停用对应策略，需人工检查后再重新启用

## 说明

- 本项目是模拟盘，不会向交易所发送真实订单
- 项目输出仅用于学习、演示与研究，不构成投资建议
